# Buddy - Highly Intelligent Chatbot

This project runs a Node.js web server serving a custom, vibrant web interface that connects to the DeepSeek API.

## Requirements
*   Node.js installed
*   A DeepSeek API Key

## Setup & Running

**The Easiest Way:**
1. Open the `C:\chatbot` folder.
2. Open the `.env` file and replace `your_api_key_here` with your actual DeepSeek API Key.
3. Double click on the **`start.bat`** file.
   *This automatically starts the website server and magically opens your web browser to the Chatbot.*

**The Manual Way:**
1. Open the `.env` file and insert your API Key.
2. Open a terminal (Command Prompt or PowerShell) and navigate into the folder:
   ```bash
   cd C:\chatbot
   ```
3. Start the server:
   ```bash
   npm start
   ```
4. Finally, open a Web Browser (like Chrome or Microsoft Edge) and go to:
   **http://localhost:3000**
